Which Build?

Release builds are recommended for most players.
They have been extensively tested and have a larger player base.

Playtest builds contain the latest improvements, but change frequently and may be broken at times.

Install OpenRA for Arch Linux

Stable releases are available in the official Arch repository, and can be installed using pacman:

$ pacman -S openra

Playtests are available via the AUR, and can be installed using yaourt:

$ yaourt openra-playtest

These packages are maintained by a third party, and may not always include the latest versions.
Please contact the downstream repository maintainers about any packaging issues.

Install OpenRA for Gentoo

Stable releases are available in the official Gentoo repositories, and can be installed using emerge:

$ emerge -av openra

Playtests are available via the “dr” overlay: 

$ layman -S -a dr -o http://github.com/cerebrum/dr/raw/master/repo.xml
$ printf '%s\n' "games-strategy/openra ~amd64" >> /etc/portage/package.keywords
$ emerge games-strategy/openra

These packages are maintained by a third party, and may not always include the latest versions.
Please contact the downstream repository maintainers about any packaging issues.

Download OpenRA source code

Follow the instructions in the INSTALL.md document to build and run OpenRA.
Visual C# Express (Windows) and MonoDevelop (OS X / Linux) are free IDEs that work with OpenRA.

If you'd like to contribute patches (or just don't want to fiddle with tar files) you can download and/or update the code using the git version control system:

$ git clone git://github.com/OpenRA/OpenRA.git
